This dataset contains a code file (run_analysis.R) which downloads the raw data set from here:

	http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

processes and manipulates the dataset and produces a tidy dataset in CSV format. (Clean_Dataset.csv)
The CodeBook.md file outlines the process the R code file follows to produce this data set.

Please note that to run this code file you will need an internet connection to download additional R libraries.

The dataset includes the following files:
=========================================

- 'README.txt'

- Clean_Dataset.csv

- Codebook.md

- run_analysis.R
